## 社会保険料の計算サンプル

## 使い方

準備
```shell
## DBにデータ投入
php gensen.php
php kenpo.php
php nenkin.php
```

社会保険料の負担額を計算する
```shell
## 計算したい時給を書く
vim tedori.php
```
計算する。
```shell
php tedori.php 
```
